"Milli the snail" is a standalone iOS & Android app built using Cocos2d/C++ and a Webgl web-app built using Phaser/Javascript

Milli was the first project i worked on at Honigstudios, I was involved as the main programmer. Besides writing code, I was tasked with helping to create the production pipeline. We decided to build the app using Cocos2D because of performance, an acceptable feature set and good community support. Bigger engines like Unity 2D were not available at the time. The code was writen in C++ and Javascript. 
The production was chalenging, mainly because of lack of mature tools that were compatible with the engine. But using a combination of 3rd party applications, a lot of creative people and some customisations to the engine source code, we created an efficient pipeline that produced a beautiful app that went on to win a Red Dot Design award.

About a year after the launch of the app Honig Studios decided to create a web-app based on Milli the snail, meant to teach kids about art.
"Milli meets art" is a web-app written in Javascript using the webgl engine Phaser.
