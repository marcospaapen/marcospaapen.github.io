"Milli the snail" is a standalone iOS & Android app and a Webgl web-app.

Milli was my first project working for Honig Studios. Besides being the main programmer writing code, I was tasked with helping to create the production pipeline. We decided to build the app using Cocos2D because of performance, an acceptable feature set and good community support. Bigger engines like Unity 2D were not available at the time. The code was writen in C++ and Javascript. 
The production was challenging, mainly because of the lack of mature tools that were compatible with the engine. By using a combination of 3rd party applications, a lot of creative people and some customisations to the engine source code, we created an efficient pipeline that produced a beautiful app that went on to win a Red Dot Design award.

About a year after the launch of the app Honig Studios decided to create a web-app based on "Milli the snail", meant to teach kids about art.
"Milli meets art" is a web-app written in Javascript using the webgl engine Phaser.

Heare is an iPhone app that helps the blind navigate using directional audio

It started out as our graduation research project at the HKU (Utrecht School of the Arts), but was later developed together with the Accessibility Foundation as an app that uses 3D directional audio to help blind people navigate by simply following the virtual position of the sound.
I was the developer in the project and worked together with 2 audio engineers to develop the system which used HRTF (head related tranfer) audio processing, this allowed us to position the sound very precisely in virtual space.
